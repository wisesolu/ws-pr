User Password Strength - Restrict Weak Password

This module helps to show user password strength by checking if the password is a combination of alphanumeric characters along with special characters and shows percentage of the strength of the password and complexity of the password with the help of few conditions.

Conditions for a strong password are:

  • Should have 8 characters.
  • Should have at least one number.
  • Should have at least one uppercase and one lowercase character.
  • Should have at least one special symbol.

You can also restrict user from creating weak password

Go to General Settings and enable Restrict Weak Password, then enter minimum password strength

If user enter a weak password then it will show a user warning and block user from creating weak password
